What is Code?

The short answer is that writing code tells the computer what to do.
Here's a longer answer: Computer code is a set of rules or instructions.
It is made up of words and numbers and when you put them in the right order it will tell your computer what you want it to do.
This page is written with code.
Have you heard about Facebook?
Facebook and all it's functionality (writing a post, comment, liking a post, etc.) is written in (computer code).

And why else?

There will be more than 800.000+ vacant jobs by 2020.
Currently there are only around 12 % of women working in IT related jobs.
Did you know? There is a project for equality which is called "50 / 50 by 2020" which creates a need for companies to employ an equal number of women in Tech / IT related jobs.
This will be a great chance to get a well paid job in tech as a woman.
Usually a tech job is well paid, including flexible working hours. There are even jobs to work from home or to work remotely.
